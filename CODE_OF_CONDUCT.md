# Code of Conduct
We are committed to fostering an inclusive, transparent environment. 
- **Respect**: Treat all researchers, regardless of career stage, with respect.
- **Transparency**: ASTRID’s reasoning loop is built on transparency; contributors are expected to uphold this in their code, documentation, and communication.
- **Collaboration**: We value constructive feedback. If you identify a flaw in the reasoning logic, propose an alternative pathway rather than just noting the error.
